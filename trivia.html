Trivia App

  • A trivia web app that allows users to play trivia games.
  • Users can view other users' profiles.
  • User average score and total games played will be updated after each game.
Current Features
  • User log in and authentication using Express Session.
  • User profiles and trivia questions stored in database.
  • Webpage rendered from server side using pug.js.
To be Added
