Crypto Watch

  • A Cryptocurrency tracking app that updates price in real time.
  • Fetch up to date data from API and renders the page with React.
  • Intuitive UI that uses graphs and charts to visualize data.
Feature Highlight
  • Automatically updates price and data for each coin.
  • Renders graphs and charts with the most recent data.
  • Use caching and concurrent calls to lower API calls
  • Minimize load time with efficient system design.
