Fitness Tracker

  • A fitness tracker which allows user to enter and save their exercises.
  • A Progressive Web App (PWA) linked with database to store and retrieve information.
  • Works on both desktop and mobile sites.
Current Features
  • Display all logged exercises in database.
  • Edit and delete each individual items.
  • Create new users and exercise logs.
  • Verify user input with Mongoose to ensure data integrity.
To be Added
  • User Authentication with JWT or Express Session, to enable profile creation.

Components Used

  • Node.js
  • Express.js
  • MongoDB
  • Mongoose
  • React.js
  • Bootstrap
  • Heroku
